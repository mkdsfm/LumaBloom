# Схема подключения KY-018 -> ESP32-C3

## Пины

- `KY-018 VCC` -> `ESP32-C3 3V3`
- `KY-018 GND` -> `ESP32-C3 GND`
- `KY-018 AO` -> `ESP32-C3 GPIO0` (ADC)

## Примечания

- Используется только аналоговый выход (`AO`).
- Для стабильности измерений подключайте короткими проводами и используйте общий GND.
- Некоторые версии KY-018 имеют маркировку `S`, `+`, `-`, где `S` — это сигнал (`AO`).

# Схема подключения BH1750 -> Waveshare ESP32-C6-LCD-1.47

## LCD на плате

Для onboard LCD проект использует пины Waveshare:

- `MOSI` -> `GPIO6`
- `SCLK` -> `GPIO7`
- `LCD_CS` -> `GPIO14`
- `LCD_DC` -> `GPIO15`
- `LCD_RST` -> `GPIO21`
- `LCD_BL` -> `GPIO22`

Контроллер дисплея: `ST7789`.

## BH1750 по умолчанию

- `BH1750 VCC` -> `3V3`
- `BH1750 GND` -> `GND`
- `BH1750 SDA` -> `GPIO20`
- `BH1750 SCL` -> `GPIO23`
- `BH1750 ADDR` -> `GND` или `NC` для адреса `0x23`

## Примечания

- Пины `GPIO20/GPIO23` выбраны в проекте как стартовая конфигурация для внешнего I2C-датчика.
- Если у вас BH1750 подключён к другим GPIO, измените `APP_BH1750_I2C_SDA` и `APP_BH1750_I2C_SCL` в `firmware/firmware_esp32c6/main/app_config.h`.
- Для I2C на коротких проводах часто достаточно внутренних pull-up, но при нестабильной линии лучше поставить внешние pull-up резисторы.
