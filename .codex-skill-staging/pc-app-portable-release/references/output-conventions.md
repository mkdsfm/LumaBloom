# Output Conventions

- Publish folder:
  `pc-app/artifacts/portable/win-x64/`
- Final zip folder:
  `pc-app/artifacts/portable/`
- Final zip filename:
  `luma-bloom-pc-app_<tag>_win-x64-portable.zip`
- Target runtime:
  `win-x64`
- Publish mode:
  self-contained
- Main project:
  `pc-app/BrightnessSensor.ConsoleApp/BrightnessSensor.ConsoleApp.csproj`
